# Backend-Cafe-Aroma-Rock
Backend Aroma Rock Café ☕💻 is a project focused on building a robust and scalable backend ⚙️ for a real coffee shop. It uses PostgreSQL 🗄️ and Express.js 🚀 to ensure secure 🔐 and efficient data handling. The system manages users 👥 and supports a points ⭐ and membership 🪪 program to boost customer loyalty.
